# Настройка пыли из-под гусениц

## 1. Компонент на танке

- Выбери объект танка (корень или объект с `TankMovement`).
- Добавь компонент **Track Dust Effect** (меню: Add Component → Tank → Track Dust Effect).

## 2. Эффект пыли в сцене

- Размести префаб/объект с эффектом пыли под гусеницами (или под корпусом).
- На объекте должен быть **Particle System** с включённым модулем **Emission** и **Rate over Time** (число можно поставить любое — скрипт будет менять его по скорости).

Рекомендации по Particle System:
- **Simulation Space**: Local (эффект двигается с танком).
- **Start Lifetime**: 0.5–1.5.
- **Start Speed**: небольшое значение (0.5–2).
- **Start Size**: под размер модели (например 0.3–1).
- **Emission → Rate over Time**: любое значение (например 10) — в игре им управляет скрипт.

## 3. Связь в инспекторе

- В компоненте **Track Dust Effect** в поле **Dust Particles**:
  - Раскрой массив (например, Size = 1 или 2).
  - Перетащи в элементы массива объекты с **Particle System** (левая/правая гусеница или один общий).

## 4. Параметры Track Dust Effect

| Поле | Назначение |
|------|------------|
| **Dust Particles** | Массив Particle System (назначаешь вручную). |
| **Min Speed For Dust** | Порог скорости (0–1), ниже которого пыль не идёт. 0.05 — почти с места. |
| **Emission Lerp Speed** | Как быстро меняется интенсивность (8 — нормально). |
| **Min Emission Rate** | Rate частиц при медленном движении (обычно 0). |
| **Max Emission Rate** | Rate при полном газе. Подбери под свой эффект (например 20–50). |

## 5. Проверка

- Запусти сцену, поезди на танке.
- При разгоне пыль должна усиливаться, при остановке — сходить на нет.
- Если пыли мало или много — меняй **Max Emission Rate** и при необходимости **Min Speed For Dust**.
